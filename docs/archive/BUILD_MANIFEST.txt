ARCHITECTURE SUMMARY
--------------------
Module System: ES6 modules throughout
- import/export for all functions
- JSON assertion for data imports
- Zero external dependencies

State Management:
- Single source: engine.gameState
- Immutable reads: renderer always reads fresh
- Update cycle: keyboard triggers engine → renderer re-renders

Keyboard Controls:
- Arrow keys: navigation
- Tab/Shift+Tab: tab switching
- Enter: select, Shift+Enter: advance day
- Escape: close modal
- Q: filter toggle (future)
- 1-7: direct tab access

Storage:
- localStorage key: 'mistwalker-save'
- Full game state serialized/deserialized
- Auto-save: every 5 minutes + on visibility change + on page unload
